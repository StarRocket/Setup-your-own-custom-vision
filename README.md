# 【Star Rocket】建立你的視覺辨識服務 Custom Vision（零基礎初學者）
近年最熱門的視覺辨識開發工具和應用廣泛的運用在各領域中，各家軟體開發品牌更是推出各類型的開源工具，為了讓更多領域可以享受到其科技應用的效率，因此我們希望透過最入門的方式，讓大家一次搞懂電腦視覺辨識，即使完全沒有程式背景的人也能一同參加。

## 課程內容
使用微軟最新的開發工具 [Azure Custom Vision Service](https://azure.microsoft.com/zh-tw/services/cognitive-services/custom-vision-service/)，帶你一步步從認識視覺辨識的原理、工具的使用、建立自己的視覺資料庫及預測模型，最後能發布成為可用的工具，基礎實作讓你邊做邊學。

什麼是 Azure Custom Vision Service（自訂視覺服務）：[微軟認知服務 Cognitive Services](https://www.customvision.ai/) 推出至今，提供許多 AI 相關服務，包含辨識、語音、語言等。強大平台透過上傳與標記就可以無痛訓練模型，不需要會寫 code 就能做出你第一個圖片辨識的 AI 服務。而 Azure Custom Vision Service 更是可以讓你輕鬆地自訂適合您獨特使用案例的最新電腦視覺模型。只要上傳一些加上標籤的影像，並讓 Custom Vision Service 執行困難的工作。讓你按一下滑鼠按鈕，就能匯出經過訓練的模型，在裝置上執行或執行為 Docker 容器，並可提供後續開發產品使用。
